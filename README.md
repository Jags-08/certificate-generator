# 📄 Certificate Generator

A simple and efficient **Certificate Generator** built to automate the creation of certificates using predefined templates. This project allows you to generate certificates in bulk using dynamic data such as names, courses, and dates.

---

## 🚀 Features

- 🎓 Automatic certificate generation  
- 📄 Customizable certificate templates  
- 🧾 Dynamic data input (Name, Course, Date, etc.)  
- 📥 Export certificates as PDF or image  
- ⚡ Bulk generation support  
- 🖥️ Easy-to-use script/interface  

---

## 🛠️ Tech Stack

- Python  
- Libraries:
  - `reportlab` (PDF generation)
  - `Pillow` (image processing)

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/Jags-08/certificate-generator.git
cd certificate-generator
Install dependencies:
pip install -r requirements.txt
▶️ Usage
Prepare your input data file (CSV/JSON):

Example CSV:

Name,Course,Date
John Doe,Python,2026-05-01
Jane Smith,AI,2026-05-01
Run the script:
python main.py
Generated certificates will be saved in the output/ folder.
🧠 How It Works
Loads a certificate template
Reads input data (CSV/JSON)
Dynamically inserts text into the template
Generates and saves certificates automatically
